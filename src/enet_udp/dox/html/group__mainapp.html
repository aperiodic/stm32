Function Documentation

int main ( void  )

Initializes a serial-over-USB CDC driver.

Activates the USB driver and then the USB bus pull-up on D+. Note, a delay is inserted in order to not have to disconnect the cable after a reset.

Activates the serial driver 6 and SDC driver 1 using default configuration.

Activates the EXT driver 1.
